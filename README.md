# AI Resume Analyzer 🚀

An AI-powered Resume Analyzer built using Python and Groq LLM.

# Features
- Upload resume
- AI analyzes resume content
- Provides intelligent suggestions for improvement
- Helps optimize resume for job applications

# Tech Stack
- Python
- Streamlit
- Groq API
- LLaMA Model

# How It Works
The application sends resume content to a Large Language Model (LLM) using the Groq API and generates smart feedback to improve job readiness.
